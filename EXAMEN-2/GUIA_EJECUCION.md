# Guía de instalación y ejecución

Ejecutar instalación de dependencias en cada proyecto:
-npm install

Levantar la base de datos (OWASP_seguro):
-npm run db:up

Verifica en Docker Desktop que la base de datos se haya creado.

Ejecutar migraciones:
-npm run migrate

Ejecutar ambos proyectos:
-npm run dev
